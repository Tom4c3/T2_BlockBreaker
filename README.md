## How to play
https://tom4c3.github.io/T2_BlockBreaker/ にアクセス<br>
- PCの場合はWASD（実際には左右にしか動かないのでADです）または矢印キーの左右でパドルを操作できます<br>
- スマホの場合は画面に触れてスライドさせることでパドルを操作できます<br>

すべてのブロックを崩すとクリアです<br>
